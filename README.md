# FirstAid HQ — Emergency Response Coordination App
ELEC1005 Assignment 2 - Daemon Hunters

## Overview
A Power Apps canvas application for the Sydney first aid community
that coordinates emergency responses, manages volunteers and tracks
incident reports in real time.

## Screens
1. Login — register and sign in
2. Dashboard — live incidents, volunteers, checklist
3. Report Emergency — submit new incident to SharePoint
4. Incidents — view all, assign self, resolve
5. Assign Volunteer — assign other volunteers to incidents
6. Volunteers — view all volunteer status
7. Roster — weekly shift schedule
8. My Profile — stats and certifications
